Skills

Yelp on Slack

Yelp on Slack is a slack application that solves the common issue of selecting a place to go to for lunch amongst teams in the workplace. The application was built using the MERN stack. Express.js and Node.js is utilized on the backend, which allows efficient data storage and fetching through custom API routes. MongoDB was used in conjunction with mongoose to provide a flexible schema in a NoSQL document-oriented database.

WanderCamp

WanderCamp is a HipCamp inspired full-stack web application that allows users to book unique experiences all over the world to cure their wanderlust. The application utilizes Ruby on Rails with RESTful APIs for storing and fetching data on the back-end, and React.js with a Redux architectural framework on the front-end. This allows for self-encapsulated React components and modular, uni-directional data flow for an efficient and satisfying user experience with scalability.

Gravity Rush

Gravity Rush is an endless side-scrolling video game written in JavaScript, where players have the ability to reverse the polarity of the world to manipulate their surroundings to survive. The goal of the game is to accrue as many points as possible before dying. As time progresses, the side-scrolling velocity will increase making it more difficult to survive.
